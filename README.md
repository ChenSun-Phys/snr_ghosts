# snr_ghosts
A Python code to compute the axion dark matter echo of supernova remnants.

Written by Manuel A. Buen-Abad and Chen Sun, 2021

Requirements
-----------------------------------------

1. Python  
2. numpy  
3. scipy  
4. astropy
5. healpy
6. astropy_healpix
7. astroquery
